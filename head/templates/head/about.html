About

Endemic Trees Monitoring System - Head Portal

A Geographic Information System for Monitoring Endemic Trees in the Philippines.

Overview

The Head Portal provides comprehensive access to all data collected by users of the Endemic Trees Monitoring System. As a head user, you can view, analyze, and generate reports from all user data, but you cannot upload or modify data.

Features

  • Interactive GIS mapping (all users' data)
  • View all map layers
  • Advanced analytics and visualizations
  • Generate comprehensive reports
  • Filter trees by species and attributes
  • View-only access to all data
  • See data from all users
  • Historical population tracking

Access Level

Head users have read-only access to all data in the system. You can:

  • View all GIS data from all users
  • Generate analytics and reports
  • View layer configurations
  • Export data for analysis

Note: Head users cannot upload data or modify existing records. This ensures data integrity and proper access control.
